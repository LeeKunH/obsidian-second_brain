### ㅡ [[이벤트핸들러-기능-용어이해-익명-콜백함수\|이벤트핸들러-기능-용어이해-익명-콜백함수]]

	- 1. 특정한 이벤트 감지하는 기능
	- 2. 특정한 이벤트 발생 후 연결된 다른 함수를 호출하는 기능 - "이벤트 핸들러 실행문의 내용을 다른 함수로 정의"

	- 비동기적인 상황 - "특정 작업의 완료 대기하지 않고 > 다음 작업 수행 시작"
